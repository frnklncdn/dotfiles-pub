* *Editor:* Neovim

== 📦 *Instalación*

. Clonar el repositorio:
+
[source, bash]
----
git clone https://github.com/frnklncdn/dotfiles-pub.git ~/.dotfiles
----
. Instalar con GNU Stow:
+
[source, bash]
----
cd ~/.dotfiles && stow */
----

== ⌨️ *Atajos Clave*
* kbd:[Mod + Enter] Abrir terminal
* kbd:[Mod + R] Reiniciar WM

== 🎨 *Apariencia*

*Crear directorio para los temas*
[source,bash]
----
mkdir ~/.local/share/{themes,icons,fonts}
----

=== *Fuentes*

*Instalación de Nerd Fonts*

* Puedes descargar cualquiera desde https://www.nerdfonts.com/["Nerd Fonts",window=read-later] y luego instalarla en tu sistema.
* Copia los directorios `JetBrainsMono` y `FontAwesome6` a `~/.local/share/fonts/` y ejecuta `fc-cache -fv`.

=== *Temas*

*Instalación de tema Flat Remix GTK Black*

* Puedes descargar cualquiera desde https://www.xfce-look.org/p/1214931["Flat Remix GTK Black",window=read-later] y luego instalarla en tu sistema.

* Descomprimir
+
[source,bash]
----
cd ~/Descargas
tar -xvf 03-Flat-Remix-GTK-Black-20240730.tar.xz
----

* Instalar
+
[source,bash]
----
mv Flat-Remix-Dark-XFWM ~/.local/share/themes/
----

=== *Iconos*

*Instalación de iconos Flat Remix ICON*

* Puedes descargarlo desde https://www.xfce-look.org/p/1012430["Flat Remix ICON theme",window=read-later].

* Descomprimir
+
[source,bash]
----
cd ~/Descargas
tar -xvf 05-Flat-Remix-Black-20251119.tar.xz
----

* Instalar
+
[source,bash]
----
mv Flat-Remix-Black-Dark ~/.local/share/icons/
----
